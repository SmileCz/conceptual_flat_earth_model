🚨 KRITICKÁ ANALÝZA

Projekt "Conceptual Flat Earth Model"

Vědecký, fyzikální a softwarově-architektonický rozbor

⚠️ VAROVÁNÍ: Tento projekt je pseudovědecká dezinformační aplikace maskovaná jako "conceptual model", kombinující sofistikované astronomické výpočty s fyzikálně nemožným modelem plochého světa.

📋 Exekutivní shrnutí

Softwarová architektura zrcadlí ideologickou podstatu: profesionální ephemeris pipelines zakomponované do fundamentálně vadného konceptu.

Klíčové zjištění:
  • Vědecky nevalidní model prezentovaný jako legitimní alternativa
  • Zneužití NASA dat (DE405), VSOP87 a dalších vědeckých zdrojů
  • Kritická porušení softwarových best practices
  • Část organizované dezinformační infrastruktury (18 jazyků)

🔬 Fyzikální a vědecká manipulace

🎭 1. Záměrná terminologická manipulace

Slovo "conceptual model" vytváří false legitimitu:

  • ✅ Legitimní: "conceptual model of atomic structure" (výuková simplifikace)
  • ❌ Tady: "conceptual flat earth" = eufemismus pro fyzikálně nemožné
// constants.js // Unitless constants for the FE dome model. // All distances are expressed as multiples of // the flat-earth disc radius (FE_RADIUS = 1)

Prezentace jako "unitless ratios" záměrně zastírá, že žádný fyzikální model nemůže vysvětlit:

  • Geodetická měření
  • Časová pásma (nemohou všechna existovat současně na disku)
  • Satelitní orbity (které projekt ironicky simuluje!)
  • Cirkapolární hvězdné stopy
  • Foucaultovo kyvadlo
  • Coriolisova síla

🔀 2. Mixování vědy a pseudovědy

Legitimní vědecké výpočty (správně implementované):
  • DE405 (NASA JPL ephemeris)

  • Ptolemaic deferent+epicycle (historický)

// feGeometry.js function celestLatLongToVaultCoord(lat, lonCelest, height, feRadius) // Projekce správných pozic planet // na VYMYŠLENOU kupoli

📐 3. "Geometrická ekvivalence" - matematický klam

| Flat / Planar | = | Globe / Spherical | | Plane diameter | = | Sphere circumference = 128,300 li | | Dome height | = | Sphere radius = 20,419.45 li | "Proof": atan(H/D) = π/2 − z = ε_globe ∴ identical elevation

Proč je to KLAM:

  1. Cherry-picking pozorování - funguje POUZE pro elevační úhly z jednoho bodu
  2. Ignoruje:
    • Horizontální vzdálenosti (fyzicky měřitelné)
    • Různé hvězdné oblohy sever/jih
    • Flight paths (budou různě dlouhé na disku)
  3. Matematická manipulace: Rovnice je tautologie pro nastavené parametry, ne důkaz fyziky
  4. Ignorování gravitace: Projekt nemá gravitační fyziku, protože by spadlo vše na pól nebo atmosféra unikla

🌑 5. Eclipse demos - ukradená věda

Projekt obsahuje: 44 solar eclipses (2021–2040) od Fred Espenak (NASA)

Espenak je vypočítal pomocí:

  • Sférické Země
  • Newtonovské gravitace
  • Relativistických korekcí

Projekt:

  1. Vezme NASA data
  2. Přemapuje na FE geometrii
  3. Tvrdí "funguje to na obou modelech"

Reality check: Saros cyklus vyžaduje 3D gravitaci Měsíce-Země-Slunce. FE nemá mechanismus pro synchronizaci lunárních fází nebo správný timing zatmění.
